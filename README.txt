This readme file doubles as a test book.

This program accepts one command line argument being the text file with pages.//

Each page in a book is delimeted by a double slash. 

Put it at the end or start of a page.// 

Try printingpress.py README.txt to make a command to write this book.

The program will prompt you for a title and author. 

Titles must be less than 16 characters.

The command will be copied to your clipboard. Very long books will require a command block to produce.//
